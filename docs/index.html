Сервер хранит только зашифрованные данные История перемещений — зашифрованные координаты за последние 24 часа, доступны только вам FCM токен — для отправки push-уведомлений о геозонах 🔐 Шифрование данных Безопасность ваших данных — наш главный приоритет. Мы применяем следующие меры защиты: 🛡️ AES-256 шифрование на стороне клиента Координаты шифруются прямо на вашем устройстве с уникальным ключом для каждой группы и случайным вектором инициализации (IV) при каждом обновлении. Сервер никогда не видит реальные координаты. Передача данных по зашифрованному каналу HTTPS Уникальный ключ шифрования для каждой группы пользователей Случайный IV (вектор инициализации) при каждом обновлении координат Правила безопасности Firebase ограничивают доступ к данным участников группы ⚙️ Как мы используем данные Отображение вашего местоположения участникам ваших групп на карте Отправка уведомлений о геозонах (входе/выходе из заданных зон) Хранение истории перемещений для последующего просмотра Аутентификация и управление учётной записью ❌ Мы никогда не продаём, не передаём и не используем ваши данные в рекламных целях. 🤝 Третьи стороны GeoTrack использует следующие сервисы третьих сторон: Firebase (Google) — аутентификация, хранение зашифрованных данных, push-уведомления.
